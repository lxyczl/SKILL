# 同义词表

## 必须掌握（出现频率最高）

| 原词 | 替换为 | 示例 |
|------|--------|------|
| show | demonstrate, indicate, reveal, suggest, confirm | The results show → The results demonstrate |
| use | utilize, employ, apply, adopt | was used to → was employed to |
| find | identify, discover, detect, observe, note | We found → We identified |
| result | outcome, finding, consequence, output | The results → The findings |
| important | crucial, vital, essential, significant, critical | is important → plays a crucial role |
| increase | enhance, augment, elevate, rise, grow | increased → experienced a rise |
| decrease | reduce, diminish, mitigate, decline, lower | decreased → underwent a decline |
| many | numerous, various, multiple, considerable, diverse | many factors → numerous factors |
| have | possess, exhibit, display, feature, demonstrate | has shown → has demonstrated |
| provide | offer, furnish, supply, deliver, yield | provides → furnishes |
