# 中文学术论文降重

一个用于中文学术论文改写降重的 Claude Code 技能。

## 功能特点

- **15种改写技巧**: 句式重组、主被动转换、拆分长句、合并短句、同义词替换等
- **3种强度级别**: 轻度、中度、重度
- **20个学科领域**: 生态水文、土木水利、绿色建筑、建筑节能等
- **专业术语保护**: 自动识别并保留专业术语、引用、公式
- **相似度检测**: 内置相似度计算器评估改写效果
- **反馈学习系统**: 自动记录改写结果，收集用户满意度，学习并优化改写策略
- **文档解析**: 自动从 docx/pdf 文件中提取文本和章节

## 快速开始

## 改写技巧

| 技巧 | 效果 | 示例 |
|------|------|------|
| 句式重组 | ★★★★★ | "研究表明X导致Y" → "Y的产生与X密切相关" |
| 主被动转换 | ★★★★★ | "研究者发现..." → "...被研究者发现" |
| 拆分长句 | ★★★★★ | 长句拆成短句，改变结构 |
| 合并短句 | ★★★★☆ | 短句合并为长句 |
| 引用位置移动 | ★★★★☆ | "X[1]" → "根据[1]，X" |
| 同义词替换 | ★★★☆☆ | "研究"→"探究/探讨" |
| 增加修饰语 | ★★★☆☆ | "研究表明" → "大量研究表明" |
| 删除冗余 | ★★★☆☆ | "进行了深入的研究" → "深入研究" |
| 调整语序 | ★★★☆☆ | "因为X所以Y" → "Y的原因在于X" |
| 添加过渡词 | ★★★☆☆ | "X。Y。" → "X。在此基础上，Y。" |

## 强度级别

| 级别 | 描述 | 适用场景 |
|------|------|---------|
| 🟢 轻度 | 主要同义词替换，不改结构 | 低相似度 |
| 🟡 中度 | 同义词+结构调整（默认） | 中等相似度 |
| 🔴 重度 | 完全重组，使用所有技巧 | 高相似度 |

## 使用建议

1. **选择合适的强度**: 根据相似度报告选择改写强度
2. **指定学科领域**: 提供学科信息以获得更准确的专业词汇替换
3. **保留专业术语**: 技能会自动保留专业术语，无需手动标注
4. **检查改写结果**: 使用相似度计算器验证改写效果
5. **提供反馈**: 使用反馈系统帮助技能学习和改进

## 注意事项

- 改写不会改变原文的核心含义和事实内容
- 专业术语、引用、公式会被自动保留
- 改写后的文本保持学术正式语体
- 不适用于非学术写作、翻译或生成全新内容
- 知网查重规则：连续13字相同算抄袭
