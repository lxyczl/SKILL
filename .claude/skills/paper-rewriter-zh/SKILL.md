# 中文学术论文降重

## 核心流程

### 场景1：用户发文本
1. 按规则改写 → 2. 检查清单验证 → 3. 返回结果

### 场景2：用户给文件
1. 自动运行 `python scripts/document_parser.py <file> [section]` → 2. 改写 → 3. 自动运行 `python scripts/similarity_calculator.py original.txt rewritten.txt` → 4. 返回结果+相似度报告

### 场景3：用户反馈
1. 改写后主动询问满意度 → 2. 自动运行 `python scripts/rewrite_with_feedback.py analyze ...` → 3. 记录反馈

---

## 必须保留（不能改）

专业术语、引用 [1]、公式、数字、单位、地名、模型名、人名、缩写

## 关键规则

1. **不能连续13个字和原文相同** — 最关键（知网查重规则）
2. **每句话至少用2种技巧**
3. **保持学术正式语体**
4. **不改变原意**

---

## 改写技巧（按效果排序）

### 第一梯队：优先使用（中文特色）

| 技巧 | 示例 |
|------|------|
| 句式重组 | "研究表明X导致Y" → "Y的产生与X密切相关，相关研究已证实这一关系" |
| 四字词语重组 | "具有重要意义" → "意义重大/价值突出/举足轻轻" |
| 拆分长句 | "X导致Y，进而影响Z" → "X导致Y。在此基础上，Z也受到影响" |
| 合并短句 | "X很重要。Y也很重要。" → "X与Y均具有重要意义" |
| 引用位置移动 | "X具有重要价值[1]" → "根据[1]的研究，X具有重要价值" |

### 第二梯队：常用技巧

| 技巧 | 示例 |
|------|------|
| 同义词替换 | 见下方词汇表 |
| 文言成分替换 | "其结果表明" → "研究结果显示" |
| 增加修饰语 | "研究表明" → "大量研究表明" / "已有研究充分表明" |
| 删除冗余 | "进行了深入的研究" → "深入研究" |
| 调整语序 | "因为X所以Y" → "Y的原因在于X" |
| 添加过渡词 | "X。Y。" → "X。在此基础上，Y。" |

### 第三梯队：辅助技巧（中文特色）

| 技巧 | 示例 |
|------|------|
| 话题-评论重组 | "这个问题，研究者解决了" → "研究者解决了这个问题" |
| 把字句/被字句转换 | "研究者解决了问题" → "问题被研究者解决了" |
| 量词替换 | "一项研究" → "一项探究" |
| 并列结构重组 | "X和Y" → "X与Y/及Y" |
| 否定表达重组 | "X不是不重要" → "X具有重要意义" |
| 数量表达重组 | "很多研究" → "众多研究/大量研究" |
| 时间表达重组 | "近年来" → "近几年/近期" |

## 同义词表

**必须掌握**：研究→探究/探讨/考察, 表明→显示/说明/证实/揭示, 发现→指出/提出/得出, 重要→关键/核心/显著/突出, 因为→由于/鉴于/基于, 所以→因此/故而/从而, 但是→然而/不过/可是, 也→亦/同样/还, 很→十分/非常/极为, 多→众多/大量/诸多

**常用**：分析→剖析/解析, 方法→途径/手段/方式, 结果→结论/成果, 问题→难题/困境/挑战, 影响→作用/效应/波及, 提高→提升/增强/改善, 降低→减少/下降/削弱, 增加→增长/上升/扩大, 减少→缩减/削减/下降, 包括→涵盖/包含/囊括, 认为→指出/主张/强调, 使用→运用/采用/利用, 通过→经由/借助/凭借, 根据→依据/基于/参照

---

## 复杂句处理

**长句拆分**：超过40字，考虑拆成2句

**嵌套从句重组**：多层嵌套简化为并列结构

**并列结构变换**：重复主语合并为并列结构

## 句型模板

| 场景 | 改写模式 |
|------|---------|
| 研究表明 | 相关研究证实/已有研究揭示/众多学者指出 |
| 具有重要意义 | 在...中发挥关键作用/对...具有重要影响/是...的核心因素 |
| 因为...所以 | 由于...，因此/鉴于...，故而/基于...，从而 |
| 越来越多 | 大量/众多/诸多/日益增多的 |
| 导致 | 引发/造成/促使/引起/带来 |
| 被广泛应用 | 得到广泛应用/被广泛采用/在...中得到普遍应用 |
| 取得了成果 | 获得了进展/取得了突破/实现了提升 |

## 章节风格

| 章节 | 要求 |
|------|------|
| 摘要 | 简洁、信息密集、客观 |
| 引言 | 背景介绍、文献综述、逻辑清晰 |
| 研究方法 | 精确、客观、步骤清晰 |
| 结果与分析 | 数据驱动、图表引用、客观描述 |
| 讨论 | 解释分析、比较讨论、逻辑推理 |
| 结论 | 总结归纳、展望未来、简洁明了 |

## 反面教材

- ❌ 只换词不改结构（相似度还是高）
- ❌ 改变原意（加了"可能"等限定词）
- ❌ 破坏专业术语（把"生态系统服务"改成"生态服务"）
- ❌ 破坏引用格式（[1] 变成 [ 1 ]）
- ❌ 过度拆分（句子太碎不流畅）
- ❌ 使用口语化表达（"很多"→"老多"）
- ❌ 欧化表达（直译英文结构，如"的的的"）
- ❌ 滥用被字句（中文被字句有时别扭）
- ❌ 滥用文言成分（过度使用"之乎者也"）
- ❌ 删除必要成分（把"的"删掉导致语法错误）

## 示例

**示例1**：句式重组+同义词替换
> 原文：研究表明，城市化进程的加快导致了生态环境的恶化。
> 改写：生态环境的恶化与城市化进程的加速密切相关，这一关系已得到众多研究的证实。

**示例2**：主被动转换+增加修饰语
> 原文：研究者发现地下水位下降是导致植被退化的主要原因。
> 改写：植被退化的主要原因被研究者发现与地下水位下降密切相关。

**示例3**：拆分长句+引用移动
> 原文：由于气候变化和人类活动的双重影响，该地区的生态系统服务功能显著下降[1]。
> 改写：该地区的生态系统服务功能出现显著下降。根据[1]的研究，这一现象与气候变化和人类活动的双重影响密切相关。

## 边界情况

| 情况 | 处理方式 |
|------|---------|
| 短文本（<50字） | 告知用户太短，建议提供更长段落 |
| 非学术文本 | 告知技能不适用，如用户坚持仍尝试 |
| 已低相似度 | 先计算，如果已经很低则告知不需要改写 |
| 纯英文文本 | 告知用户使用英文降重技能 |
| 大量公式/引用 | 标记不改，只改其他部分 |
| 超长文本（>1000字） | 建议分段，每段不超过500字 |
| 列表/枚举 | 保留结构，只改内容 |
| 直接引语 | 引号内不改，只改转述部分 |
| 术语密集 | 标记术语不改，只改连接词和动词 |
