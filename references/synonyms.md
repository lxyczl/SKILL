# 中文同义词表

## 必须掌握（出现频率最高）

| 原词 | 替换为 | 示例 |
|------|--------|------|
| 研究 | 探究、探讨、考察、分析、剖析 | 本研究 → 本探究/本文 |
| 表明 | 显示、说明、证实、揭示、指出 | 研究表明 → 研究显示/研究证实 |
| 发现 | 指出、提出、得出、揭示、确认 | 研究发现 → 研究揭示/研究指出 |
| 重要 | 关键、核心、显著、突出、重大 | 具有重要意义 → 发挥关键作用 |
| 因为 | 由于、鉴于、基于、出于 | 因为X → 由于X/鉴于X |
| 所以 | 因此、故而、从而、于是 | 所以Y → 因此Y/故而Y |
| 但是 | 然而、不过、可是、只是 | 但是Z → 然而Z/不过Z |
| 也 | 亦、同样、还、又 | 也有 → 亦有/同样具有 |
| 很 | 十分、非常、极为、相当 | 很重要 → 十分重要/极为关键 |
| 多 | 众多、大量、诸多、丰富 | 很多研究 → 众多研究/大量研究 |
