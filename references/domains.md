# 中文学科专业词汇表

## 计算机科学与人工智能

### 专业术语（不改）
深度学习、卷积神经网络、循环神经网络、Transformer、注意力机制、自然语言处理、计算机视觉、强化学习、迁移学习、联邦学习、生成对抗网络、扩散模型、大语言模型、预训练、微调、特征提取、端到端、损失函数、反向传播、梯度下降

### 替换词
- 深度学习 → 深层学习、深度神经网络方法
- 卷积神经网络 → CNN、卷积网络
- 循环神经网络 → RNN、递归神经网络
- 注意力机制 → 注意力模型、注意力分配机制
- 自然语言处理 → NLP、文本智能处理
- 计算机视觉 → 机器视觉、视觉智能
- 强化学习 → 奖励学习、增强学习
- 迁移学习 → 知识迁移、域适应方法
- 生成对抗网络 → GAN、对抗生成模型
- 大语言模型 → LLM、大规模语言模型
- 预训练 → 预先训练、前置训练
- 微调 → 精调、细调
- 特征提取 → 特征抽取、特征获取
- 端到端 → 端对端、一体化
- 损失函数 → 代价函数、目标函数
- 反向传播 → 误差回传、BP算法
- 梯度下降 → 梯度优化、参数更新
